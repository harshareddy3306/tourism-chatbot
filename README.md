# Tourism Management Chatbot

A simple rule-based chatbot built using Python to respond to tourism-related questions.

## Description
The chatbot uses keyword matching to identify user intent and provide predefined responses. It demonstrates basic chatbot logic and user interaction using the command line.

## Technologies Used
- Python

## How to Run
1. Run the program:
   python chatbot.py

## Use Case
Useful for understanding chatbot fundamentals and conversational flow.
